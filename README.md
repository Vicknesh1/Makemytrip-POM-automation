# Makemytrip-POM-automation
Selenium automation framework for MakeMyTrip flight booking using Java, Maven, TestNG, and Page Object Model (POM).

This project is an end-to-end automation framework built using Selenium WebDriver, Java, Maven, and TestNG, following the Page Object Model (POM) design pattern.

It automates the flight booking flow on MakeMyTrip.com
, performing actions like:

Navigating to the Flights section

Entering source and destination

Selecting a travel date for the next month

Searching for flights

Identifying and printing the cheapest and second cheapest flights

Opening a new browser tab and navigating to Google

Includes an additional scenario for extended automation validation

The framework demonstrates:

Modular Page Object design

Clean and maintainable code structure

Test execution through TestNG

Maven build and dependency management
